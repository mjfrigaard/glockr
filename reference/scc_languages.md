# List languages supported by scc

List languages supported by scc

## Usage

``` r
scc_languages()
```

## Value

A
[`tibble::tibble()`](https://tibble.tidyverse.org/reference/tibble.html)
with one row per language and columns `language` and `extensions`.

## Examples

``` r
if (FALSE) { # \dontrun{
scc_languages()
} # }
```
