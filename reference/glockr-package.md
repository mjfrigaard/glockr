# glockr: Count Lines of Code Using 'scc'

An R wrapper around the `scc` (Succinct Code Counter) command-line
utility. Counts lines of code, comments, and blank lines across many
languages, with optional complexity metrics.

## Details

`scc` must be installed and on the system PATH. Download it from
<https://github.com/boyter/scc>.
